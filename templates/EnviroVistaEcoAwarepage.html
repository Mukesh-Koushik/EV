AVAILABLE TASKS

Task name : Home Energy Audit, 5 GC Coins

Description : Conduct a thorough inspection of your home’s energy use to identify areas where you can improve efficiency. This includes checking for air leaks, inspecting insulation, and evaluating the efficiency of heating and cooling systems. Benefits : Reduces energy consumption, lowers utility bills, and decreases carbon footprint.

Task name : Install Solar Panels, 5 GC Coins

Description : Install solar panels on your roof or property to harness renewable energy from the sun. This task may involve choosing the right system, getting permits, and professional installation. Benefits : Provides a clean energy source, reduces reliance on fossil fuels, and can lower electricity costs.

Task name : Create a Compost Bin, 5 GC Coins

Description : Set up a compost bin in your backyard to recycle organic waste like kitchen scraps and yard waste into nutrient-rich compost. Benefits : Reduces landfill waste, produces free fertilizer for your garden, and lowers methane emissions from landfills.

Task name : Upgrade to Energy-Efficient Appliances, 5 GC Coins

Description : Replace old, energy-guzzling appliances with energy-efficient models certified by programs like ENERGY STAR. Benefits : Saves energy, reduces greenhouse gas emissions, and lowers energy bills

Task name : Install a Rainwater Harvesting System, 5 GC Coins

Description : Install a system to collect and store rainwater for use in irrigation, gardening, and other non-potable applications. Benefits : Conserves water, reduces water bills, and provides a sustainable water source

Task name : Switch to LED Lighting, 5 GC Coins

Description : Replace incandescent or fluorescent light bulbs with energy-efficient LED bulbs throughout your home. Benefits : Uses less energy, lasts longer, and reduces energy costs.

Task name : Plant a Native Garden, 5 GC Coins

Description : Create a garden using native plants that are adapted to your local climate and require minimal water and maintenance. Benefits : Supports local biodiversity, reduces water use, and provides habitat for wildlife.

Task name : Install Low-Flow Fixtures, 5 GC Coins

Description : Replace existing faucets, showerheads, and toilets with low-flow models to reduce water consumption. Benefits : Lowers water use, reduces water bills, and conserves water resources.

Task name : Implement a Home Recycling Program, 5 GC Coins

Description : Set up a system for sorting and recycling different types of waste, including plastics, metals, paper, and glass. Benefits : Reduces landfill waste, conserves natural resources, and promotes recycling habits.

Task name : Build a DIY Solar Oven, 5 GC Coins

Description : Construct a solar oven using simple materials like cardboard, aluminum foil, and glass to cook food using solar energy. Benefits : Uses renewable energy, reduces electricity or gas usage, and provides an eco-friendly cooking method.

Task name : Create a Green Roof, 5 GC Coins

Description : Install a layer of vegetation on your roof to improve insulation, reduce stormwater runoff, and provide a habitat for wildlife. Benefits : Enhances energy efficiency, reduces heat island effect, and promotes biodiversity.

Task name : Set Up a Bicycle Commuting Station, 5 GC Coins

Description : Create a dedicated area for storing bicycles and related gear to encourage biking as a sustainable mode of transportation. Benefits : Reduces carbon emissions, promotes physical health, and decreases traffic congestion.

Task name : Transition to a Plant-Based Diet, 5 GC Coins

Description : Incorporate more plant-based meals into your diet, reducing the consumption of animal products. Benefits : Lowers your carbon footprint, conserves water, and supports animal welfare.

Task name : Install Smart Home Devices, 5 GC Coins

Description : Install smart thermostats, lighting controls, and other devices to optimize energy use in your home. Benefits : Enhances energy efficiency, reduces utility bills, and offers convenient control of home systems.

Task name : Organize a Community Clean-Up, 5 GC Coins

Description Plan and execute a local clean-up event to pick up litter, remove debris, and beautify public spaces. Benefits : Improves community aesthetics, reduces pollution, and fosters community spirit.
